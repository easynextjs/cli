> Depends on Cursor, Vercel, next-ui, Supabase

Easy-to-Use CLI for Next.js

## Getting Started

```bash
# Install the CLI globally
$ npm install -g @easynext/cli
$ yarn global add @easynext/cli
$ pnpm add -g @easynext/cli
```

## Commands

```bash
# Check tools installation
$ easynext doctor

# set language
$ easynext lang <'ko' | 'en'>
```
