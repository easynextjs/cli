Easiest way to start Next.js project ⚡️ [한글](./README.ko.md)

## Installiation

```bash
# (recommended) use npm
$ npm install -g @easynext/cli

# if you are using yarn, pnpm
$ yarn global add @easynext/cli
$ pnpm add -g @easynext/cli
```

## Available commands

```bash
# Check tools installation
$ easynext doctor

# Set language
$ easynext lang <'ko' | 'en'>
```

## Coming Soon Commands

```bash
# Create a new project
$ easynext create <project-name>

# Set google auth
$ easynext auth google|apple|kakao|naver

# Init next-ui
$ easynext next-ui

# Init supabase
$ easynext supabase

# Add Google Sheet modules
$ easynext google-sheet
```
